CDN

PushFeedback offers a simple and efficient way to gather user feedback directly from your website. Follow the steps below to integrate the PushFeedback widget into a standard HTML website.

Prerequisites Ensure you have access to the source code of your website, and you're comfortable editing HTML files.

  1. Load the PushFeedback Widget

To load the widget, include the following lines in the <head> section of your HTML file:

<!-- Load the PushFeedback widget CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/pushfeedback/dist/pushfeedback/pushfeedback.css">

<!-- Load the PushFeedback widget script -->
<script type="module" src="https://cdn.jsdelivr.net/npm/pushfeedback/dist/pushfeedback/pushfeedback.esm.js"></script>
  1. Integrate the Feedback Button Choose a location on your website where you'd like the feedback button to appear. This could be in the footer, sidebar, or any other section.

Insert the following code snippet at your chosen location:

<feedback-button project="<YOUR_PROJECT_ID>">Send feedback</feedback-button>

Important: Replace <YOUR_PROJECT_ID> with the project ID provided by PushFeedback. This ensures that feedback is correctly associated with your project.

  1. Save and Test After adding the code snippets:

  2. Save your HTML file.

  3. Open the website in a web browser to verify the feedback button appears and functions correctly.

Final Thoughts Regularly checking the feedback gathered can offer invaluable insights about areas of improvement or features your users might want. By integrating PushFeedback, you're making a step towards understanding your users better.
